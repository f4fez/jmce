Philips/NXP LPC764 microprocessor.

LPC764 is one Philips single chip microcontroller compatible with Intel 8051, with internal ROM. One good reason to use it is that is one of the few chip available in dual inline case. The machine cycle is only 6 clock compared with the 12 of the original intel implementation.
